# AI Investment Assistant 💰

An AI-powered personal investment assistant for tracking mutual funds, index funds, SIPs, portfolio performance, and long-term financial planning.

## Features

- 📈 Portfolio Dashboard
- 💰 SIP Tracker
- 📊 Investment Analytics
- 📉 Profit & Loss Tracking
- 📅 Investment History
- 🎯 Goal Planner
- 🤖 AI Investment Assistant
- 🔔 Smart Reminders
- 📄 PDF Reports
- 🌙 Dark Mode
- 📱 Responsive UI

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
- APScheduler
